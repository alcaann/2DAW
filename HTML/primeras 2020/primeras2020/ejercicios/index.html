Atributos

Todos los elementos HTML pueden tener atributos
 Los atributos proporcionan información adicional acerca de un elemento
 Siempre se especifican en la etiqueta de inicio
 Por lo general vienen en pares de nombre / valor como: nombre = "valor"

El atributo lang El lenguaje del documento puede ser declarada en el html etiqueta. El lenguaje se declara con el lang atributo. La declaración de una lengua es importante para las aplicaciones de accesibilidad (lectores de pantalla) y motores de búsqueda
